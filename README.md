# Aveno - اپلیکیشن مدیریت کار و ردیابی سیگار

یک اپلیکیشن React Native کاملاً افلاین برای مدیریت کارهای روزانه و ردیابی مصرف سیگار با رابط کاربری فارسی و تم دارک مینیمال.

## ویژگی‌ها

### مدیریت کارها
- ✅ ایجاد، ویرایش و حذف کارها
- ✅ دسته‌بندی کارها با رنگ و آیکون
- ✅ تنظیم تاریخ با تقویم شمسی فارسی
- ✅ فعالیت مستمر/غیرمستمر
- ✅ تنظیم تایم و ددلاین
- ✅ سیستم پاداش/مجازات (امتیازدهی)
- ✅ اولویت‌بندی کارها (کم، متوسط، زیاد)
- ✅ فیلتر و جستجو
- ✅ تکمیل سریع کارها
- ✅ نمایش کارهای گذشته از موعد

### ردیابی سیگار
- ✅ شمارش تعداد سیگار روزانه (افزودن/کاهش)
- ✅ تنظیم حد مجاز روزانه
- ✅ نمودار دایره‌ای پیشرفت روزانه
- ✅ نمودار خطی روند هفتگی
- ✅ نمودار میله‌ای مصرف ماهانه
- ✅ گزارش ماهانه و روزانه
- ✅ آمار کامل (میانگین، حداکثر، حداقل، streak)
- ✅ نمایش درصد استفاده

### گزارش‌گیری

## تکنولوژی‌ها

- React Native 0.81.5
- Expo SDK 54
- TypeScript
- WatermelonDB (SQLite)
- Zustand (State Management)
- React Navigation
- React Native Paper
- Victory Native (Charts)
- Bottom Sheet Modals

## نصب و راه‌اندازی

1. نصب وابستگی‌ها:
```bash
npm install
```

2. اجرای اپلیکیشن:
```bash
npm start
```

3. اجرا روی Android:
```bash
npm run android
```

## ساختار پروژه

```
src/
├── components/       # کامپوننت‌های قابل استفاده مجدد
├── screens/         # صفحات اصلی
├── navigation/      # تنظیمات ناوبری
├── store/           # Zustand stores
├── database/        # WatermelonDB schema و models
├── repositories/    # Repository pattern
├── services/        # Business logic
├── utils/           # Helper functions
├── types/           # TypeScript types
└── theme/           # تم دارک و استایل‌ها
```
